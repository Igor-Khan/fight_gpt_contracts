# Sample Hardhat Project

Here is `FightGPT` contracts.

ENV vars:
```
PRIVATE_KEY=YOUR_PRIVATE_KEY
```

Try running some of the following tasks:

```shell
npx hardhat help
npx hardhat test
REPORT_GAS=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts --network testnet
```
